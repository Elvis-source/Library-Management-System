# Online Library Management System

This project is a simple Online Library Management System implemented in C++ using Object-Oriented Programming (OOP) concepts.  
It was developed as part of a research-based assignment to practice designing classes and implementing basic system functionality.

## Features implemented
- Add books to the library
- Register users
- Borrow and return books
- The basic validation of borrowing and returning books.

## OOP Concepts Used
- Classes and Objects
- Encapsulation
- Abstraction

## Testing
The tests included in the main() function cover:
  -Successful borrowing of a book
  -Attemting to borrow an already borrowed book
  -Returning a borrowed book


## How to Compile and Run the Program
1. Open a terminal
2. Compile the program using:
   g++ library.cpp -o library
3. Run the program using:
   ./library
